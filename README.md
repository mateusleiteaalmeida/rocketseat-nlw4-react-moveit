## 🚀 Iniciando

Clone o projeto e acesse a pasta

```bash
$ git clone https://github.com/guilhermecapitao/nlw04-moveit-web.git && cd nlw04-moveit-web
```

Siga os passos abaixo
```bash
# Instalar as dependências
$ yarn

# Iniciar o projeto
$ yarn dev
```
